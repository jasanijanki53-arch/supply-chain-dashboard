# Supply Chain Dashboard - Why First Class Ships Last

## Overview
This Power BI dashboard analyzes delivery mode performance across markets,
identifying why First Class shipping has the highest late delivery rate (95.3%).

## Key Metrics
- Total Orders: 65.8K
- Total Sales: $36.78M
- Total Profit: $3.97M
- Overall Late Delivery Rate: 54.8%

## Shipping Mode Performance
| Ship Mode      | Late Rate |
|----------------|-----------|
| First Class    | 95.3%     |
| Second Class   | 76.5%     |
| Same Day       | 45.5%     |
| Standard Class | 38.1%     |

## How to Use
1. Download `supply chain.pbix`
2. Open in Power BI Desktop
3. Explore filters by Market, Category, and Ship Mode

## Tools Used
- Power BI Desktop
- DAX
